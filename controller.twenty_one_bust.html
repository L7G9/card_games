Controller for model.twenty_one_bust.game.Game and console.

Coordinates model and text view (print, input, get_option & clear_screen) to run a game of 21 Bust for the user.

play_game()¶

Plays the game.

Deals cards and loops through each player until each had completed their turn.

player_turn(player)¶

Process a player’s turn.

Loops through player’s actions until their turn is complete.

Parameters:

player – The Player instance who’s turn it is.

reset_game(winners)¶

Prepare game to play again.

Update the game instance ready to play again. Display the new order in which the players will take their turn.

Parameters:

winners – A list of player instances who won the last game.

resolve_game()¶

Provides feedback to user on the results of the game.

Updates the game instance now all players have had their turn. Displays the contents of each players hand and the total when sicking or if they went bust, then displays who won this game.

run()¶

Enters main loop for the game of 21 Bust.

Performs initiaL Setup, plays game then prompts user to play again. Then depending on their response, resets the game and repeats or prints each player’s win count.

setup()¶

Perform initial setup of game.

Asks user for their name and how many players they would like to play against. Creates player instances for each of those and provides feedback.

user_player_actions(player)¶

Process a user controlled player’s action.

Prompts user to stick to twist, updates game instance with their choice and provides feedback.

Parameters:

player – The Player instance who’s turn it is.
