class model.card_game.deck.Deck(name: str, values: Value, suits: Suit)¶

Bases: CardGroup

Class to represent a standard deck of cards.

A deck of cards with no jokers.

cards¶

A list of Card instances.

name¶

A name describing this deck.

add_deck(values: Value, suits: Suit)¶

Add deck of Cards.

Creates all the Card objects for one standard deck of cards and adds it to this deck. Can be used multiple times for larger deck sizes.

Parameters:

  • values – An Enumeration of the card values in the deck.

  • suits – An Enumeration of the card suits in the deck.

deal(number_of_cards: int, players: list[model.card_game.player.Player])¶

Deal Cards from this Deck.

Removes cards from the top of this Deck and adds them to the end CardGroup one by one until each CardGroup have the same number of Cards.

Parameters:

  • number_of_cards – An integer equal to the number of cards each Player’s hand should receive.

  • players – A list of Players to receive the cards.

return_cards(card_group: CardGroup)¶

Returns Cards to Deck.

Adds all the Cards in the CardGroup face down. Then removes the Cards from the CardGroup.

Parameters:

card_group – A CardGroup object to return the cards from.
